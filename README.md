# Web Scraper

Scrapes bestbuy for AMD cpus and writes to a `cpus.txt` file.

To run (NPM):

```
npm install
```

This will install all the necessary files into your node_modules folder from the package.json

```
npm run dev
```

to run the program.

to run (YARN):

```
yarn
```

This will install all the necessary files into your node_modules folder from the package.json

```
yarn dev
```

to run the program.
